# ML_applications

機械学習・ディープラーニングを活用した実戦的なビジネス応用プロジェクト集です。
回帰、分類、クラスタリング、時系列予測、自然言語処理など、多岐にわたる手法を用いて具体的な課題解決に取り組んでいます。

## プロジェクト一覧

### 1. 商品の売上予測（回帰・時系列）
- **概要**: 過去の日付ごとの売上データを分析し、来月の売上個数を予測します。
- **活用シーン**: 在庫の最適化、発注管理の効率化。

### 2. スパムメール分類（自然言語処理）
- **概要**: テキストデータをベクトル化し、高度な分類アルゴリズムを用いてスパムメールを自動判定します。
- **活用シーン**: セキュリティ対策、メールフィルタリング機能。

### 3. レビュースコア予測（感情分析）
- **概要**: ユーザーのレビューコメント（テキスト）を分析し、内容からレビュースコアを予測します。
- **活用シーン**: 顧客満足度の定量的把握、フィードバックの自動分類。

### 4. 高齢者のクラスター分析（ヘルスケア）
- **概要**: 地域の高齢者データを元にセグメンテーションを行い、特性に応じた健康促進アプローチを提案します。
- **活用シーン**: 地方自治体の健康増進施策、個別化された医療支援。

### 5. 顧客セグメンテーション（マーケティング）
- **概要**: 年齢、性別、年収などの属性データからお客様を最適なグループに分類します。
- **活用シーン**: ターゲットを絞ったプロモーション、顧客理解の深化。

### 6. 顧客離脱予測（ニューラルネットワーク）
- **概要**: NN（Neural Network）を用いて、口座解約の兆候がある顧客を早期に予測します。
- **活用シーン**: チャーンレート（離脱率）の改善、リテンションマーケティング。

### 7. アパレル売上高予測（経営戦略）
- **概要**: 業界特有のトレンドや過去データを分析し、売上高を予測して戦略立案を支援します。
- **活用シーン**: 販売計画の策定、店舗展開の意思決定支援。

### 8. 株価推移予測（LSTM・ディープラーニング）
- **概要**: 時系列データに強い「LSTM」を活用し、複雑な株価の変動パターンを学習・予測します。
- **活用シーン**: 資産運用補助、市場トレンドの分析。
