# データサイエンス・実践プロジェクト集


学習したプロジェクトの記録です。基礎から応用まで、機械学習の主要アルゴリズムと前処理（EDA）の実践をカバーしています。

### プロジェクト一覧
1.  **タイタニック号の生存予測**（二値分類・入門）
    - 生存か死亡かを予測。欠損値処理とカテゴリ変数のエンコーディングを学習。
2.  **自社商品の購買顧客予測**（マーケティング分析・分類）
    - 属性データから購買確率を予測。ビジネス指標への変換を実践。
3.  **心臓病のリスク予測**（ヘルスケア・分類）
    - 医療データを用いたリスク判定。不均衡データの扱いや特徴量の寄与度を確認。
4.  **生徒の次回テスト点数予測**（単回帰/重回帰分析）
    - 数値の連続的な変化を予測。相関分析と回帰モデルの評価指標を学習。
5.  **顧客のグループ分け（クラスタリング）**（教師なし学習）
    - 教師なし学習によるセグメンテーション。K-means法などを用いた顧客属性の抽出。
6.  **顧客セグメントの多クラス分類**（多クラス分類）
    - 3つ以上のクラス（セグメント）を予測。決定木やランダムフォレストの活用。
7.  **中古車の価格予測**（複雑な回帰分析）
    - 多様な特徴量を持つデータからの価格推定。外れ値処理と特徴量エンジニアリング。
8.  **従業員の離職予測**（生存分析/分類）
    - 誰が辞める可能性が高いかを分析。人事領域（HR Tech）におけるデータ活用。
9.  **商品の売上予測**（時系列・回帰）
    - 過去のトレンドから将来の数値を予測。トレンド要因と季節性の把握。
10. **糖尿病患者の予測**（臨床統計・分類）
    - 身体検診データを用いた発症リスク予測。高い精度が求められる医療分野の課題。
11. **データサイエンティストの給料予測**（回帰・実データ分析）
    - 職種やスキルの価値を定量化。データから市場価値を導き出す実践的分析。

### 使用技術
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **Scikit-learn** (機械学習アルゴリズム全般)
- **Git / GitHub** (バージョン管理・プロジェクト管理)
